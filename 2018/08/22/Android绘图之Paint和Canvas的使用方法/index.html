Android绘图之Paint和Canvas的使用方法

  • Canvas
  • View布局

在自定义 View 的时候经常会使用到 Paint 和 Canvas 进行绘图，这些类比较像一些工具类，主要集中在方法的理解与使用。下面是给出的一些常用方法，当然还有很多还有方法没有列出来，使用的时候可以直接查看源码查找相应的方法。

Paint类

setStyle

用于设置画笔的风格，可以指定是圆心还是实心，该方法在矩形，圆形有明显的效果

//参数style为画笔的风格
public void setStyle(Paint.Style style)

可选参数包括：

  • Style.FILL: 实心
  • Style.FILL_AND_STROKE: 同时显示实心和空心
  • Style.STROKE: 空心

setStrokeCap

用来控制我们的画笔在离开画板时候留下的最后一点图形，或者说叫做笔触风格 ，笔触就像我们现实世界中的笔，如果你用圆珠笔在纸上戳一点，那么这个点是个圆，它代表了笔的笔触形状，如果我们把一支铅笔笔尖削成方形的，那么画出来的线条会是一条弯曲的“矩形”，这就是笔触的意思。

public void setStrokeCap(Cap cap)

参数可选列表：

  • Cap.BUTT : 无笔触风格
  • Cap.ROUND ：圆形
  • Cap.SQUARE ： 方形

setStrokeJoin

设置线段连接处样式

public void setStrokeJoin(Join join)

可选值包括：

  • Join.MITER（结合处为锐角）
  • Join.Round(结合处为圆弧)
  • Join.BEVEL(结合处为直线)

setTypeface

用于设置画笔的字体样式，可以使用系统自带的字段，也可以使用自定义的字体

//typeface为字体样式
public void Typeface(Typeface typeface)

一些可选值包括：

  • Typeface.DEFAULT: 默认字体
  • Typeface.DEFAULT_BOLD: 加粗字体
  • Typeface.MONOSPACE: monospace字体
  • Typeface.SANS_SERIF: sans字体
  • Typeface.SERIF: serif字体

setColor

用于设置画笔的颜色

//参数color为颜色值，也可以使用Color类定义的颜色
public void setColor(int color)

下面是一些Color类定义的颜色：

  • Color.BLACK:黑色
  • Color.BLUE:蓝色
  • Color.CYAN:青绿色
  • Color.DKGRAY:灰黑色
  • Color.YELLOW:黄色
  • Color.GRAY:灰色
  • Color.GREEN:绿色
  • Color.LTGRAY:浅绿色
  • Color.MAGENTA:红紫色
  • Color.TRANSPARENT:透明色

setAlpha

用于设置画笔的透明度，为透明度，其取值范围为0~255，数值越小越透明

public void setAlpha(int a)

setPathEffect

详解Paint的setPathEffect(PathEffect effect)

setStrokeWidth

用于设置画笔的空心线宽，该方法在矩形，圆形，等图形上有明显的效果；

//参数width为线宽，浮点型数据
public void setStrokeWidth(float width)

setARGB

用于设置画笔的颜色和透明度

public void setARGB(int a,int r,int g,int b);
  • 参数a为透明度，范围0~255
  • 参数r为红色的颜色值，范围0~255
  • 参数g为绿色的颜色值，范围0~255
  • 参数b为蓝色的颜色值，范围0~255

setTextSize

用于设置画笔的字体大小，主要用于绘制字符串

// 参数为字体大小
 public void setTextSize(float textSize)

setShadowLayer

用于设置添加阴影

public void setShadowLayer(float radius, float dx, float dy, int shadowColor)
  • adius 表示阴影的倾斜度
  • dx:水平方向位移的像素
  • dy:垂直方向位移的像素
  • shadowColor:设置阴影的颜色

setTextScaleX

设置字体水平方向的拉伸，默认为1，当大于1时表示向两边拉伸，小于1时表示向中间压缩

public void setTextScaleX(float scaleX)

setFakeBoldText

用于设置文字是否为粗体

public void setFakeBoldText(boolean fakeBoldText)

setStrikeThruText

设置是否带有删除线效果

public void setStrikeThruText(boolean strikeThruText)

setUnderlineText

用于设置画笔的下划线，当取值为true时，表示显示下划线

public void setUnderlineText(Boolean underlintext)

setTextAlign

设置文字对齐方式

// 参数为文字对齐方式
public void setTextAlign(Align align)

可选参数包括：

  • Align.LEFT
  • Align.RIGHT
  • Align.CENTER

Canvas保存和还原

Canvas提供了几个方法，让我们可以方便的对Canvas的状态进行更改和还原。这些方法是：

  • save()
  • restore()
  • restoreToCount(int saveCount)。

我们在对 Canvas 进行平移、旋转、放大等操作时候，可以调用save()方法，将当前修改过的Canvas状态进行保存，调用restore() 方法后，会将 Canvas 还原成最近的一个save() 的状态。

save()方法还会有一个返回值，我们也可以调用 restoreToCount(int saveCount) 方法，将这个返回值作为参数传递进去，就可以将Canvas还原成某一个特定的save()状态。

一个可参考操作：

 canvas.translate(100,100); // 平移（100,100）
 int save1 = canvas.save(); // 保存Canvas状态（状态1）
 canvas.scale(2, 2); // 放大2倍
 int save2 = canvas.save(); // 保存Canvas状态（状态2）
 canvas.restore(); // 返回最新的save状态，即状态2
 canvas.restoreToCount(save1);// 手动指定的返回到 状态1
